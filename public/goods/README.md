### 2. 토트백

## 이미지 업로드 방법

1. 이미지 파일을 준비하세요
2. 파일명을 위에 명시된 이름으로 변경하세요
3. 해당 폴더에 파일을 복사하세요

예시:
- `public/goods/mugs/mug_goodvibes_main.jpg`
- `public/goods/tote-bags/tote_goodvibes_main.jpg`
- `public/goods/notebooks/notebook_goodvibes_main.jpg`

## 참고

이미지가 없을 경우 빈 이미지나 플레이스홀더가 표시될 수 있습니다.
실제 이미지를 업로드하면 자동으로 표시됩니다.
