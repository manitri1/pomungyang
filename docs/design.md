# 🎨 트랜스미디어 캐릭터 플랫폼 UI/UX 디자인 가이드

**Role:** Senior UI/UX Designer
**Style:** Modern
**Mood:** 재미있고, 신세대 취향, 애니메이션 스타일

---

## 🧭 1. Design System Overview

| 항목          | 설명                                                            |
| ----------- | ------------------------------------------------------------- |
| **디자인 방향성** | 트랜스미디어 캐릭터 세계관을 중심으로, 시각적 몰입감과 브랜드 개성을 강화하는 “신세대 애니메이션 톤” 디자인 |
| **핵심 가치**   | ✨ 재미 + 🎭 참여감 + 💬 커뮤니티성                                      |
| **UI 원칙**   | 단순한 인터랙션, 생동감 있는 마이크로 애니메이션, 반응형 인터페이스                        |
| **타깃 유저층**  | 10~30대 청소년, 대학생, 청년층, 반려동물 애호가                                |

**디자인 톤앤매너 키워드:**

* 현대적, 카툰-스타일 그래픽, 경쾌한 모션
* 볼드한 타이포그래피 + 라운드 코너 UI
* 색상 대비를 통한 가독성 확보

## 🎨 2. Color Palette (for TailwindCSS)

| Category           | Name            | Hex       | Tailwind Variable        | 설명                               |
| ------------------ | --------------- | --------- | ------------------------ | -------------------------------- |
| **Primary**        | Electric Violet | `#7B61FF` | `--color-primary-500`    | 캐릭터 브랜드의 핵심 컬러, 메인 CTA 버튼/링크에 사용 |
| **Secondary**      | Neon Coral      | `#FF6F61` | `--color-secondary-500`  | 감정적 포인트(이벤트, 알림)에 사용             |
| **Accent**         | Mint Green      | `#00E5A0` | `--color-accent-500`     | 보조 강조색, 포인트 라벨 및 강조 텍스트          |
| **Neutral**        | Warm Gray       | `#F4F4F4` | `--color-neutral-100`    | 기본 배경 및 카드 베이스                   |
| **Text Primary**   | Charcoal        | `#2B2B2B` | `--color-text-primary`   | 본문 텍스트                           |
| **Text Secondary** | Slate Gray      | `#6E6E6E` | `--color-text-secondary` | 보조 텍스트 및 설명문                     |
| **Highlight**      | Lemon Yellow    | `#FFF95B` | `--color-highlight`      | 캐릭터 네임/아이콘 하이라이트                 |

**색상 조합 근거:**
보색 조합(Complementary) 구조로 **보라(Electric Violet)** ↔ **코랄(Neon Coral)** 대비를 주어 역동적이고 에너제틱한 인상 형성.
이는 신세대 감성(‘짙은 보라 + 네온’)과 잘 맞음.

## 🔄 5. Interaction Patterns

| 유형                   | 설명               | 예시            |
| -------------------- | ---------------- | ------------- |
| **Hover Motion**     | 캐릭터 카드 배경색 전환    | 보라 → 민트 변화    |
| **Scroll Animation** | 섹션 진입 시 페이드인     | Hero → 이벤트 구간 |
| **Button Feedback**  | 클릭 시 살짝 튀는 애니메이션 | CTA 버튼        |
| **Modal Transition** | 팝업 열릴 때 슬라이드업    | 캐릭터 상세보기      |
| **Loading State**    | 스켈레톤 UI          | 데이터 지연 시      |

## 📐 6. Breakpoints

```scss
$breakpoints: (
  'mobile': 320px,
  'tablet': 768px,
  'desktop': 1024px,
  'wide': 1440px
);
```

| 구분                      | 설명                 |
| ----------------------- | ------------------ |
| **Mobile (≤768px)**     | 1열 카드 구조, 햄버거 메뉴   |
| **Tablet (768–1024px)** | 2열 그리드, Sidebar 접힘 |
| **Desktop (≥1024px)**   | 3열 고정, Sidebar 노출  |
| **Wide (≥1440px)**      | Hero 풀스크린, Grid 확대 |
