# 🧾 PRD 문서 – 트랜스미디어 콘텐츠 캐릭터 플랫폼

---

## 1. 📘 제품 개요 (Detailed Product Description)

**제품명:** 트랜스미디어 캐릭터 브랜딩 플랫폼
**목적:**
다양한 미디어(웹툰, 일러스트, 이벤트, 팝업스토어 등)를 통해 **캐릭터의 세계관과 감성을 확장**하고, 사용자 참여형 브랜딩 경험을 제공하는 웹 기반 플랫폼입니다.

**핵심 가치 제안:**

* 캐릭터 IP를 중심으로 한 팬 커뮤니티 및 브랜딩 허브 구축
* 오프라인 이벤트(팝업스토어, 스탬프 투어)와 온라인 스토리텔링의 융합
* MZ세대(특히 반려동물 및 감성 콘텐츠에 관심 있는 층) 대상의 지속적 참여 유도

---

## 2. 🔍 레퍼런스 서비스 및 분석 (Reference Services with Rationale)

| 참고 서비스                     | 설명                                  | 적용 근거                         |
| -------------------------- | ----------------------------------- | ----------------------------- |
| **라인 프렌즈 (LINE FRIENDS)**  | 캐릭터 기반 IP 확장, 오프라인 팝업 및 굿즈 판매 중심    | 브랜딩 일관성, 팬 커뮤니티 중심의 경험 구조 참고  |
| **카카오프렌즈 (KAKAO FRIENDS)** | 각 캐릭터의 페르소나와 스토리를 기반으로 한 팬 몰입형 콘텐츠  | 캐릭터별 스토리텔링 구조 및 이벤트 마케팅 방식 반영 |
| **산리오 (Sanrio Japan)**     | 다수의 캐릭터를 통합 관리하며, 팬층별 맞춤형 전시/이벤트 기획 | 복수 캐릭터 운영 모델 및 팬 세분화 전략 참고    |
| **펫프렌즈/펫박스**               | 반려견 커뮤니티와 캐릭터 감성 결합                 | 반려견 애호가 중심의 타깃 사용자 감성 접근 참고   |

**요약 근거:**
이 플랫폼은 **“캐릭터 세계관 + 팬 경험 + 실물 체험”**의 통합 모델을 지향하므로,
디지털 콘텐츠와 오프라인 경험을 동시에 관리하는 **하이브리드 IP 플랫폼** 모델을 차용합니다.

## 3. ⚙️ 핵심 기능 및 명세 (Core Features and Specifications)

| 기능                    | 설명                                    | 사용자 가치                  |
| --------------------- | ------------------------------------- | ----------------------- |
| **① 캐릭터 페르소나 소개**     | 각 캐릭터의 성격, 세계관, 관계도, 특징 시각화           | 사용자 몰입감 향상, 브랜드 친밀도 강화  |
| **② 캐릭터 챌린지**         | 각 캐릭터별 미공개 웹툰, 일러스트, 퀘스트 콘텐츠 제공       | 스토리 참여형 팬 경험, 지속적 방문 유도 |
| **③ 팝업스토어**           | 체험존(포토존, AR 체험) 및 판매존(굿즈, 한정판 아이템) 안내 | 오프라인 참여 유도 및 매출 전환      |
| **④ 캐릭터 투어 (스탬프 투어)** | 지정된 장소 방문 후 디지털 스탬프 획득                | 지역 연계형 팬활동 및 수집형 재미     |
| **⑤ 굿즈 소개 및 판매**      | 캐릭터별 굿즈 카탈로그 및 구매 링크 제공               | IP 수익화 기반, 팬 리텐션 강화     |

## 5. 👤 사용자 페르소나 및 시나리오 (User Persona and Scenarios)

### 🎯 주요 타깃: 10대~30대, 감성 콘텐츠 및 반려견 애호가

| 구분                  | 특징                  | 주요 행동           | 니즈             |
| ------------------- | ------------------- | --------------- | -------------- |
| **청소년 (10~19세)**    | 웹툰, 일러스트, 캐릭터 감성 선호 | SNS 공유, 챌린지 참여  | 재미, 표현 욕구, 소속감 |
| **대학생 (20~24세)**    | 이벤트 참여 및 캐릭터 수집욕 강함 | 팝업스토어 방문, 굿즈 구매 | 취미와 정체성 결합 콘텐츠 |
| **청년 직장인 (25~35세)** | 감성 굿즈 및 반려견 캐릭터 선호  | 캐릭터 IP 소비 및 공유  | 감정 치유, 스트레스 해소 |
| **반려견 애호가**         | 캐릭터-반려견 공감 콘텐츠에 관심  | SNS 참여형 챌린지     | 반려동물 중심 콘텐츠 욕구 |

---

### 📖 사용자 시나리오 예시

**시나리오 1: 캐릭터 팬의 웹 방문**

1. 사용자는 메인 페이지에서 캐릭터 소개 섹션을 탐색
2. 좋아하는 캐릭터 선택 → 스토리/일러스트 감상
3. 캐릭터 챌린지 참여 → SNS 공유 → 다음 이벤트 예약

**시나리오 2: 오프라인 이벤트 참여자**

1. 팝업스토어 위치 정보 확인
2. 현장 스탬프 투어 참여 후 디지털 인증 완료
3. 한정 굿즈 구매 및 이벤트 인증 리워드 획득

## 6. 🧰 기술 스택 제안 (Technical Stack Recommendations)
