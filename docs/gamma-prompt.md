# Gamma 프레젠테이션 프롬프트

## 🎯 Gamma에 입력할 프롬프트

```
트랜스미디어 캐릭터 브랜딩 플랫폼 "PoMungYang (포멍양)"의 앱 소개 및 사용 설명서를 만들어줘.

## 프로젝트 개요
- 앱 이름: PoMungYang (포멍양)
- 성격: 트랜스미디어 캐릭터 브랜딩 플랫폼
- 목적: 포항 지역 연계 캐릭터 IP를 통한 팬 경험 제공 및 지역 관광 활성화
- 주요 캐릭터: 청멍이 (승천을 꿈꾸는 용의 후예), 백냥이 (재물과 효율을 중시하는 현실주의 고양이)

## 주요 기능 및 페이지 구조

### 1. 캐릭터 소개 (/characters)
- 청멍이, 백냥이 등 캐릭터별 프로필
- 캐릭터의 페르소나, 스토리, 일러스트 소개
- 각 캐릭터의 세계관 및 특징 설명

### 2. 세계관 만들기 (/characters/{id}/worldview)
- 캐릭터별 세계관 게시판
- 팬들이 스토리와 이미지를 공유하는 커뮤니티
- 청멍이 세계관, 백냥이 세계관 별도 운영
- 게시글 작성, 이미지/영상 업로드 가능

### 3. 챌린지 (/challenges)
- 캐릭터와 함께하는 참여형 챌린지 목록
- 각 챌린지의 미션 및 보상 정보

### 4. 이벤트 (/challenges/events)
- 특별 이벤트 목록 및 상세 정보
- 이벤트 참여 인증 사진 업로드
- 예시: "청멍이와 함께하는 승천 기원: 운제산 용오름 챌린지", "백냥이의 포항 펫-득 미식 로드 챌린지"
- 이벤트 상세 페이지: 개요, 컨셉 스토리, 참여 방법, 상품, 공지사항, 해시태그

### 5. 팝업스토어 (/popup)
- 오프라인 팝업스토어 위치 및 정보
- 체험존 (포토존, AR 체험) 안내
- 판매존 (굿즈, 한정판 아이템) 정보

### 6. 캐릭터 투어 (/tour)
- 포항 지역 연계 스탬프 투어
- 지도 기반 투어 코스 안내
- 스탬프 수집 현황 및 리워드 시스템

### 7. 굿즈샵 (/goods)
- 캐릭터 굿즈 카탈로그
- 상품 상세 정보 및 구매 링크
- 예시 상품: 머그컵, 토트백, 노트북, 폰케이스, 펜던트, 뱃지 세트 등

### 8. 마이페이지 (/mypage)
- 사용자 정보 관리
- 내 스탬프 수집 현황
- 참여한 챌린지 및 이벤트 기록

### 9. 로그인/회원가입 (/auth)
- 아이디/비밀번호 기반 로그인
- 회원가입 기능
- NextAuth.js 기반 인증 시스템

### 10. 관리자 페이지 (/admin)
- 이벤트 관리: 이벤트 등록, 수정, 삭제
- 굿즈 관리: 상품 추가, 수정, 삭제
- 컨텐츠 아이디어 생성: AI 기반 컨텐츠 아이디어 생성 (Gemini API 활용)

## 기술 스택
- 프레임워크: Next.js 15 (App Router)
- 언어: TypeScript
- 스타일링: Tailwind CSS
- UI 컴포넌트: Shadcn UI
- 데이터베이스: Supabase
- 인증: NextAuth.js
- 배포: Vercel

## 디자인 컨셉
- 모던하고 깔끔한 UI
- 캐릭터 컬러를 활용한 브랜딩
- 반응형 디자인 (모바일/태블릿/데스크톱 지원)
- 직관적인 네비게이션 (사이드바 기반)

## 사용자 여정 (User Journey)
1. 홈페이지 접속 → 메인 히어로 섹션과 최신 이벤트 티저 확인
2. 회원가입/로그인
3. 캐릭터 소개 페이지에서 관심 있는 캐릭터 탐색
4. 세계관 만들기에서 팬 커뮤니티 참여
5. 챌린지/이벤트 페이지에서 활동 참여
6. 인증 사진 업로드 및 공유
7. 스탬프 투어로 포항 지역 탐방
8. 굿즈샵에서 상품 구매
9. 마이페이지에서 활동 현황 확인

## 특별 기능
- 실시간 데이터 동기화 (Supabase 기반)
- 이미지/영상 업로드 기능
- 이벤트 참여 갤러리
- 스탬프 수집 게임화 요소
- AI 기반 컨텐츠 아이디어 생성 (관리자 기능)

## 타겟 사용자
- 포항 지역 관광객 및 주민
- 캐릭터 팬 커뮤니티
- 반려동물을 키우는 사람들 (이벤트 참여 대상)
- 지역 관광 활성화에 관심 있는 사람들

## 프레젠테이션 구성 제안
1. 커버 슬라이드: PoMungYang 브랜드 소개
2. 앱 개요 및 비전
3. 주요 캐릭터 소개 (청멍이, 백냥이)
4. 핵심 기능 상세 설명 (각 페이지별 스크린샷 및 설명)
5. 사용자 여정 플로우차트
6. 관리자 기능 소개
7. 기술 스택 및 아키텍처
8. 로드맵 및 향후 계획

시각적으로 매력적이고 이해하기 쉬운 프레젠테이션을 만들어줘. 각 기능별로 실제 사용 화면을 상상해서 설명하고, 사용자가 어떻게 앱을 활용할 수 있는지 명확하게 보여줘.

### 3단계: 커스터마이징

- 실제 스크린샷이나 목업 이미지 삽입
- 색상 테마를 캐릭터 브랜딩에 맞게 조정

### 4단계: 추가 정보 입력 (선택사항)
다음 정보를 추가로 제공하면 더 정확한 프레젠테이션을 만들 수 있습니다:

```
## 추가 상세 정보

### 캐릭터 상세
- 청멍이: "승천을 꿈꾸는 마지막 용의 후예", 포항 구릉포의 아홉 마리 용 전설에서 영감, 청룡이 되고 싶어함, 이상과 희망의 상징
- 백냥이: "재물과 효율을 중시하는 현실주의 고양이", "돈이 최고!"라는 모토, 현실적이고 실용적인 캐릭터, 가성비/가심비 분석 전문가

### 주요 이벤트 예시
1. 운제산 용오름 챌린지: 반려동물과 함께 포항 최고봉 운제산 등정 및 승천 기원 사진 촬영
2. 펫-득 미식 로드: 백냥이 추천 포항 펫 프렌들리 맛집 방문 및 인증

### 지역 연계 포인트
- 포항 운제산 (해발 482m)
- 구릉포 해변 (아홉 마리 용 전설의 장소)
- 포항 펫 프렌들리 맛집
- 지역 굿즈 및 관광 상품 연계

### 커뮤니티 기능
- 세계관 게시판: 팬들이 자유롭게 스토리와 이미지 공유
- 이벤트 갤러리: 참여 인증 사진 공유
- 좋아요 및 댓글 기능
- SNS 공유 기능

## 🎨 프레젠테이션 제작 팁

1. **시각적 요소 강화**
   - 캐릭터 일러스트 활용
   - 실제 앱 스크린샷 삽입
   - 플로우차트 및 다이어그램 활용

2. **스토리텔링**
   - 사용자 여정을 스토리로 구성
   - "청멍이와 함께하는 모험" 같은 서사적 접근

3. **인터랙티브 요소**
   - 각 슬라이드에 호버 효과
   - 클릭 가능한 링크나 버튼 시연

4. **데이터 시각화**
   - 기능별 사용자 수/참여율 (있는 경우)
   - 스탬프 수집 통계
   - 이벤트 참여 현황

5. **CTA (Call to Action)**
   - 각 기능 설명 후 "지금 바로 체험해보세요" 같은 행동 유도
