# 🧭 정보 구조 문서 (IA Document)

**프로젝트명:** 트랜스미디어 캐릭터 브랜딩 플랫폼
**내비게이션 타입:** Sidebar
**인증 방식:** Required (회원 로그인 기반 접근)

---

## 1. 🗺️ 사이트 맵 (Site Map)

```
홈 (/)
├── 로그인 / 회원가입 (/auth)
│   ├── 로그인
│   └── 회원가입
├── 캐릭터 소개 (/characters)
│   ├── 캐릭터 리스트
│   └── 캐릭터 상세 페이지 (/characters/:id)
│        ├── 페르소나
│        ├── 스토리/웹툰
│        └── 일러스트 갤러리
├── 챌린지 (/challenges)
│   ├── 진행 중 챌린지
│   └── 완료한 챌린지
├── 팝업스토어 (/popup)
│   ├── 체험존 안내
│   └── 판매존 정보
├── 캐릭터 투어 (/tour)
│   ├── 투어 지도
│   └── 스탬프 수집 현황
├── 굿즈샵 (/goods)
│   ├── 굿즈 리스트
│   └── 상품 상세 (/goods/:id)

│   ├── 내 스탬프
│   ├── 내 챌린지
│   └── 설정

     ├── 공지사항
     └── FAQ
```

## 2. 🔄 사용자 흐름 (User Flow)

### [1] 신규 사용자 (비회원 → 회원)

1. 홈 접속 → 로그인 페이지 이동
2. 회원가입 폼 작성 (이메일, 비밀번호, 닉네임, 관심 캐릭터 선택)
3. 가입 완료 → 자동 로그인 후 캐릭터 소개 페이지 진입

### [2] 로그인 사용자

1. 로그인 후 → Sidebar를 통해 캐릭터별 메뉴 탐색
2. 챌린지 참여 시 → 진행 상태가 LocalStorage에 저장
3. 팝업스토어/투어 페이지에서 → QR 또는 스탬프 인증
4. 마이페이지에서 → 수집 현황 및 리워드 확인

### [3] 관리자 플로우 (선택적 확장)

1. 로그인 → 관리자 전용 메뉴 진입
2. 신규 캐릭터 및 콘텐츠 등록 → 미리보기
3. 이벤트 관리 및 사용자 피드백 확인

| 사이드바   | 팝업스토어     | 로그인 필요 | 오프라인 체험 및 판매 정보  |

## 5. 📂 콘텐츠 구성 (Content Organization)

| 콘텐츠 유형        | 구성 요소                 | 설명                 |
| ------------- | --------------------- | ------------------ |
| **캐릭터 정보**    | 이름, 이미지, 성격, 세계관, 관계도 | 각 캐릭터별 고유 페르소나 중심  |
| **웹툰 / 일러스트** | 에피소드, 작가 정보, 업로드 일자   | 트랜스미디어 콘텐츠 중심      |
| **팝업스토어 정보**  | 위치, 일정, 체험존/판매존 정보    | 오프라인 이벤트 연결        |
| **투어 / 스탬프**  | 장소명, QR코드, 획득 여부      | 지역 기반 체험 연동 콘텐츠    |
| **굿즈 정보**     | 썸네일, 이름, 가격, 구매 링크    | 외부 결제 시스템 연동       |
| **사용자 데이터**   | 스탬프 현황, 챌린지 진행률       | LocalStorage 기반 관리 |

## 6. 🧩 인터랙션 패턴 (Interaction Patterns)

| 인터랙션 유형                          | 설명                          | UX 고려사항                   |
| -------------------------------- | --------------------------- | ------------------------- |
| **탭 전환 (Tab Switching)**         | 캐릭터 상세 내 스토리/일러스트/페르소나 간 이동 | 전환 시 애니메이션으로 몰입감 강화       |
| **스탬프 수집 (Click/QR)**            | 투어 지점 도착 후 QR 스캔 → 스탬프 획득   | 성공 시 피드백 애니메이션 제공         |
| **챌린지 참여 (Modal/Progress Bar)**  | 미션 수행 시 진행률 실시간 업데이트        | LocalStorage 저장 + 시각적 피드백 |
| **굿즈 보기 (Hover/Modal)**          | 마우스 오버 시 상세 정보 노출           | 반응형 카드 UI 유지              |
| **푸시 알림 (Browser Notification)** | 신규 이벤트 알림                   | 사용자 허용 설정 제공              |

## 7. 🌐 URL 구조 (URL Structure)

| 페이지     | URL 예시            | 설명            |
| ------- | ----------------- | ------------- |
| 홈       | `/`               | 기본 진입 페이지     |
| 로그인     | `/auth/login`     | 사용자 로그인       |
| 회원가입    | `/auth/register`  | 계정 생성         |
| 캐릭터 리스트 | `/characters`     | 전체 캐릭터 목록     |
| 캐릭터 상세  | `/characters/:id` | 캐릭터 개별 페이지    |
| 챌린지     | `/challenges`     | 진행 중 / 완료 챌린지 |
| 팝업스토어   | `/popup`          | 오프라인 이벤트 정보   |
| 투어      | `/tour`           | 스탬프 지도 및 현황   |
| 굿즈샵     | `/goods`          | 상품 리스트        |
| 상품 상세   | `/goods/:id`      | 개별 상품 상세      |
| 마이페이지   | `/mypage`         | 사용자 개인 정보     |
| 고객센터    | `/support`        | FAQ, 공지사항     |

**SEO 고려사항:**

* URL은 모두 **영문 소문자 + 하이픈(-)** 사용
* 각 페이지별 **meta title / description / OG tags** 설정
* `robots.txt` 및 `sitemap.xml` 자동 생성
